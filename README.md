## Description ##

Outreach and CCB API integration.

## Installation ##

### Manual Installation ###

1. Upload the entire `/lqd-outreach` directory to the `/wp-content/plugins/` directory.
2. Activate Liquid Outreach through the 'Plugins' menu in WordPress.
